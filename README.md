# Aethera Studio - Building the Eternal

A cinematic landing page for Aethera Studio - building platforms for brilliant minds, fearless makers, and thoughtful souls.

## Features

- **Cinematic Video Background**: Custom fade-in/fade-out loop with smooth transitions
- **Premium Typography**: Instrument Serif for display, Inter for body text
- **Fully Responsive**: Optimized for all devices
- **SEO Optimized**: Comprehensive meta tags, Schema.org structured data, sitemap, and robots.txt
- **Smooth Animations**: Custom fade-rise animations for a polished experience
- **Demo Banner**: Professional demo label linking to Hubber Web

## Sections

1. **Hero** - Full-screen video with compelling headline
2. **Studio** - Creative vision and what we build
3. **About** - Company story and stats
4. **Journal** - Blog/articles section
5. **Contact** - CTA section

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React
- Google Fonts (Instrument Serif, Inter)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
